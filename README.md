# Viral Launch Pad 🚀

This folder contains the **Active Launch System**.
Phase 1: Monitor & Asset Gen. Phase 2: Deployment.

## What's Inside
- **Viral Monitor**: Scans key accounts (Elon, Trump, Vitalik) for viral sparks.
- **Asset Generator**: Automatically creates landing pages (`index.html`) for detected tickers.
- **Token Launcher**: *(Coming Soon)* Will create Solana wallet, deploy to Pump.fun, and snipe supply.

## How to Run

**Start Monitor (Continuous):**
```bash
python run_launch.py
```
*Select Option 1 to start monitoring the feed.*

**Outputs:**
- `outputs/` folder: Contains generated landing pages.
- **Telegram Bot**: Sends real-time alerts.
- **Google Sheet**: Logs new signals to `Viral_Monitor` tab.
